# Taskflow Frontend

React + TypeScript + Redux Toolkit frontend for the realtime task collaboration backend.

## Commands

```bash
pnpm --filter @taskflow/frontend dev
pnpm --filter @taskflow/frontend typecheck
pnpm --filter @taskflow/frontend lint
pnpm --filter @taskflow/frontend test
```

## Environment

Copy `.env.example` to `.env`:

```bash
cp apps/frontend/.env.example apps/frontend/.env
```
